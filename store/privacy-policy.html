FormEase - Privacy Policy

Last updated: April 9, 2026

Overview

FormEase is a Chrome extension that helps you fill web forms automatically using a locally stored profile. We are committed to protecting your privacy. This policy explains how your data is handled.

Data Collection

FormEase does not collect, transmit, or store any data on external servers.

  • No analytics or tracking scripts
  • No cookies or fingerprinting
  • No network requests to any server
  • No user accounts or authentication

Data Storage

All profile data (name, email, phone, address, etc.) is stored exclusively in your browser's local storage using the chrome.storage.local API. This data:

  • Never leaves your device
  • Is accessible only to the FormEase extension
  • Can be exported/imported as a JSON file by you at any time
  • Can be deleted by removing the extension

Permissions

FormEase requests the following Chrome permissions:

  • activeTab — To detect form fields on the current page
  • storage — To save your profile locally
  • scripting — To inject the form-filling script into web pages

These permissions are the minimum required for the extension to function. No additional permissions are requested.

Third-Party Services

FormEase does not use any third-party services, APIs, SDKs, or libraries that collect user data.

Children's Privacy

FormEase does not knowingly collect information from children under 13.

Changes to This Policy

If this policy is updated, the changes will be reflected on this page with an updated date.

Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.

FormEase - 개인정보 처리방침

최종 수정일: 2026년 4월 9일

개요

FormEase는 로컬에 저장된 프로필을 사용하여 웹 폼을 자동으로 채워주는 크롬 확장 프로그램입니다. 사용자의 개인정보를 보호하기 위해 최선을 다합니다.

데이터 수집

FormEase는 어떤 데이터도 외부 서버로 수집, 전송, 저장하지 않습니다.

  • 분석 도구나 추적 스크립트 없음
  • 쿠키나 핑거프린팅 없음
  • 서버로의 네트워크 요청 없음
  • 사용자 계정이나 인증 없음

데이터 저장

모든 프로필 데이터(이름, 이메일, 전화번호, 주소 등)는 브라우저의 로컬 저장소 (chrome.storage.local API)에만 저장됩니다. 이 데이터는:

  • 사용자의 기기를 절대 벗어나지 않습니다
  • FormEase 확장 프로그램만 접근할 수 있습니다
  • 언제든지 JSON 파일로 내보내기/가져오기 할 수 있습니다
  • 확장 프로그램을 삭제하면 함께 삭제됩니다

권한

FormEase는 다음의 크롬 권한을 요청합니다:

  • activeTab — 현재 페이지의 폼 필드를 감지하기 위해
  • storage — 프로필을 로컬에 저장하기 위해
  • scripting — 웹 페이지에 폼 채우기 스크립트를 주입하기 위해

제3자 서비스

FormEase는 사용자 데이터를 수집하는 제3자 서비스, API, SDK, 라이브러리를 사용하지 않습니다.

문의

개인정보 처리방침에 관한 문의는 GitHub 리포지토리의 이슈를 통해 해주시기 바랍니다.
