# FormEase - 원클릭 폼 자동 채우기

매번 같은 이름, 이메일, 전화번호를 입력하는 데 지치셨나요?
FormEase는 한 번 프로필을 저장하면, 어떤 웹사이트의 폼이든 원클릭으로 자동 채워줍니다.

## 주요 기능

- **원클릭 자동 채우기** — 이름, 이메일, 전화번호, 주소 등 20개 필드를 한 번에 입력
- **키보드 단축키** — Ctrl+Shift+F로 즉시 채우기, Ctrl+Shift+E로 미리보기 (Mac: Cmd)
- **되돌리기** — 잘못 채웠으면 되돌리기 버튼 한 번으로 원래 값 복원
- **자동 저장** — 프로필 편집 시 자동으로 저장. 저장 버튼 안 눌러도 됩니다
- **내보내기/가져오기** — JSON 파일로 프로필 백업 및 다른 브라우저로 이동

## 왜 FormEase인가?

- **계정 없음** — 회원가입 필요 없습니다
- **서버 없음** — 모든 데이터는 내 브라우저에만 저장됩니다
- **완전 무료** — 숨겨진 요금 없습니다
- **AI 없음** — 가볍고 빠릅니다. 인터넷 연결 없이도 동작합니다

## 이런 분께 추천합니다

- 매일 여러 사이트에 같은 정보를 입력하는 분
- 구직 활동 중 수십 개의 지원서를 작성하는 분
- 개인정보가 외부 서버로 전송되는 것이 불안한 분

## 지원 필드

이름, 성, 성명, 이메일, 전화번호, 생년월일, 성별, 우편번호, 주소, 상세주소, 도시, 회사, 부서, 직책, 경력, 자기소개, 웹사이트, GitHub, LinkedIn

한국어와 영어 폼을 모두 인식합니다.
